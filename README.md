# 🪪 SwiftUI Profile Card

A SwiftUI project building a polished profile card UI using `VStack`, `HStack`, and `ZStack` — a practical exercise in SwiftUI layout, image styling, and composing reusable view components.

---

## 🤔 What this is

This project demonstrates how to compose a visually appealing profile card using SwiftUI's layout stacks. It covers `ZStack` for layered backgrounds, `HStack` for row-based content, circular image clipping with `.clipShape(Circle())`, and text hierarchy using `font` and `foregroundColor` modifiers.

## ✅ Why you'd use it

- **Stack layout** — `VStack`, `HStack`, `ZStack` used together in a real UI
- **Circular avatar** — `Image` with `.clipShape(Circle())` and `.overlay` border
- **Custom card style** — `RoundedRectangle` background, shadow, padding
- **Composable views** — shows how to break a UI into smaller, reusable SwiftUI views

## 🚀 Getting Started

### 1. Clone the Repo
```bash
git clone https://github.com/NDCSwift/SwiftUIProfileCard.git
cd SwiftUIProfileCard
```

### 2. Open in Xcode
- Double-click `SwiftUIStacks.xcodeproj`

### 3. Set Your Development Team
In Xcode: **TARGET → Signing & Capabilities → Team**

### 4. Update the Bundle Identifier
Change `com.example.MyApp` to a unique identifier (e.g., `com.yourname.ProfileCard`).

---

## 🛠️ Notes

- If you see a code signing error, check that Team and Bundle ID are set.

## 📦 Requirements

- iOS 16+
- Xcode 15+
- Swift 5.9+
